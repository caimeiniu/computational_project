# Reference result tables

These are compact, already-generated result summaries used by the project
report. Re-running scripts writes new artifacts under `../../outputs/`; it does
not overwrite these reference tables.

- `selection_results_summary.csv`: main 20-200-label learning curves.
- `all_data_10pc_linear_regression_baseline.csv`: all-data 10-PC reference.
- `extended_budget_summary.csv`: learning curves through 1,000 labels.
- `pca_10_to_100_baseline_results.csv`: all-data PCA-dimension sweep.
