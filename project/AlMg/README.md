# Al(Mg) grain-boundary segregation: code and data

Reproducibility package for the **Al(Mg)** part of the report
*"Training-Site Selection and Hybrid Monte Carlo for Grain-Boundary
Segregation Beyond the Dilute Limit"* (Chen Cheng, Fu Jiayi, Caimei Niu).

This directory contains only the scripts and data needed to reproduce the
Al(Mg) figures, Table II, and the spectrum/mechanism numbers. The
point-selection (Chen Cheng) and Pt(Au) (Fu Jiayi) parts of the report are
not included here.

The test: the independent-site Fermi-Dirac (FD) model is built from the same
single-solute segregation-energy spectrum and compared against finite-
concentration hybrid Monte Carlo / MD (HMC). For Al(Mg) the HMC GB occupation
falls below the canonical closed-box FD prediction once Mg-Mg interactions on
the boundary matter.

Pipeline scripts that build the inputs above: `generate_polycrystal.py` (200 A,
16-grain Al box) -> `gb_identify.py` (a-CNA -> `gb_mask_200A.npy`) ->
`sample_delta_e.py` (`delta_e_results_n500_200A_tight.npz`) ->
`fit_delta_e_spectrum.py`. HMC runs use `build_fdseed_inits.py` +
`make_gb_group_include.py` and the LAMMPS decks in `data/decks/`.

## Current numbers (source of truth = the JSON files)

Al(Mg) HMC vs canonical FD at X_c=0.10 (Table II):

| T (K) | X_GB^FD | X_GB^HMC | 95% CI | gap | source JSON |
|---|---|---|---|---|---|
| 500 | 0.3519 | 0.1858 | [0.1857, 0.1859] | -0.1661 | `hmc_T500_Xc0.10_fdseed_resume12.json` |
| 700 | 0.2956 | 0.2064 | [0.2061, 0.2066] | -0.0892 | `hmc_T700_Xc0.10_fdseed_targeted_resume11.json` |
| 800 | 0.2734 | 0.2012 | [0.2011, 0.2014] | -0.0722 | `hmc_T800_Xc0.10_fdseed_targeted_resume10.json` |

The HMC values are post-burn-in means of X_GB(t) with a 95% moving-block
bootstrap CI. Trajectories are seeded at the FD prediction and descend below
it; they are descending upper bounds, not fully converged equilibria, so the
last digit should not be over-read. The from-below (random-seeded) branches
`hmc_T{700,800}_Xc0.10_random_targeted_resume7.json` (X_GB = 0.1985, 0.2036)
are included as the lower side of the two-initial-condition bracket.

Spectrum validation (`compare_vs_wagih_200A_tight.json`): our n=500 sample has
skew-normal fit (mu, sigma, alpha) = (+6.34, 20.06, -1.47) kJ/mol vs the Wagih
reference (+6.72, 20.84, -1.40); two-sample KS D = 0.026, p = 0.892.

## Reproduce

```bash
cd project/AlMg
python -m pip install -r requirements.txt
```

The figure/analysis path needs only `numpy`, `scipy`, and `matplotlib`. Each
generator script takes explicit `--out-png` / `--out-json` paths; run with
`-h` for arguments. Re-running the full atomistic pipeline additionally needs
LAMMPS built with the MANYBODY, MC, and VORONOI packages, plus the EAM
potential in `data/potentials/Al-Mg.eam.fs`.

## Not included (large or external)

- Raw LAMMPS dumps, restart files, and final `.lmp` snapshots (~61 MB each,
  ~1.3 GB total). These are intermediate; the figures above are reproduced from
  the post-processed JSONs and the small snapshots in `data/`. They can be
  regenerated by re-running `generate_polycrystal.py` + the decks in
  `data/decks/`.
- The Wagih *et al.* (2020) reference spectrum used by `compare_vs_wagih.py`:
  download from Zenodo (doi:10.5281/zenodo.4107058).
