# Au(Pt) 3D Segregation Results

Final comparison files are in `final_bulk50_vs_raw_wagih/`.

Key run:
- Alloy: Au solvent with Pt solute, written as `Au(Pt)`
- Sample size: 500 GB sites
- Bulk references: 50 far-from-GB bulk sites
- Wagih comparison: raw nonzero Au(Pt) Zenodo segregation energies

Important files:
- `aupt_3d_bulk50_deltae_results.csv`: our 500 calculated Delta E values
- `wagih_aupt_raw_deltae_nonzero.csv`: raw Wagih Au(Pt) reference data with zero entries removed
- `final_bulk50_vs_raw_wagih/aupt_3d_bulk50_raw_wagih_deltae_analysis.xlsx`: Excel summary
- `final_bulk50_vs_raw_wagih/aupt_3d_bulk50_raw_wagih_deltae_histogram_vs_wagih.png`: histogram comparison
- `final_bulk50_vs_raw_wagih/aupt_3d_bulk50_raw_wagih_ranked_deltae.png`: ranked Delta E plot
- `final_bulk50_vs_raw_wagih/aupt_3d_bulk50_raw_wagih_summary.csv`: numerical summary and KS comparison

Old Cu/Ni and Ni(Cu) generated outputs were moved to the local ignored `archive/` folder.
